# DSTD-GCN

This is the official implementation of DSTD-GCN [(Learning Constrained Dynamic Correlations in Spatiotemporal Graphs for Motion Predictions)](https://arxiv.org/abs/2204.01297).

This paper is currently under review of IEEE Transactions of Neural Networks and Learning Systems. We will release the codes if our paper is finally accepted.
